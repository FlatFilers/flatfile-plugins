# Flatfile Record Hook Plugin

This plugin exports a function named `recordHook` that sets up an event listener for the `commit:created` event. The purpose of this plugin is to provide a concise syntax for running custom logic on individual data records in Flatfile.

## Install & import

```bash
npm i @flatfile/plugin-record-hook @flatfile/hooks
```

See the full README in the [https://https://flatfile.com/docs/plugins/transform/record-hook](Flatfile Docs)
